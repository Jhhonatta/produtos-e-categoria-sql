## 👨🏻‍💻 Executando o projeto ou o teste

Utilize o **yarn** ou o **npm install** para instalar as dependências do projeto.
Em seguida, inicie o projeto ou o teste.

Iniciando a versão web:

```cl
yarn dev
```

Iniciando o test:

```cl
yarn test
```
